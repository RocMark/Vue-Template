# Vue-Template

## 前言
VueCli 3.0

## 使用
```
  // 建立專案環境
  npm i 

  // 修復高危套件
  npm audit fix
  
  // 啟動 Vue 專案
  npm run serve
  
  // 開啟 json-server
  npm run server
  npm run json-server
```

## 功能
- 全域 SCSS 檔案
- 全域 axios (建議只在 store.js 使用 axios)
- JSON Server
- Git Commit 模板
- BluePrint 藍圖建立檔案
