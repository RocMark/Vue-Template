# Vue-Template

## 前言
創建寫 side project 用的模板
[測試網址(停用中)](https://rocmark.github.io/Vue-Template/.)

## 使用
```
  // 建立專案環境
  npm i 
  
  // 啟動 Vue 專案
  npm run serve
  
  // 開啟 json-server
  npm run json-server
```

## 功能
- 全域 SCSS 檔案
- 全域 axios
- JSON Server
- Git Commit 模板
- BluePrint 藍圖建立檔案
