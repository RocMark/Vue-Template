# Vue-Template


## 使用
- VueCli 3.0
```
  // 安裝 vue cli 套件
  npm install -g @vue/cli

  // 安裝 json-server
  npm install -g json-server

  // 建立專案環境
  npm i 

  // 修復高危套件 (有提示才需要)
  npm audit fix
  
  // 啟動 Vue 專案
  npm run serve
  
  // 開啟 json-server
  npm run server
  npm run json-server
```

## 功能
- 全域 SCSS 檔案
- 全域 axios
- JSON Server
- Git Commit 模板
- BluePrint 藍圖建立檔案
- Vuex 模組化
- route 模組化
- Global Mixin
- 區域 Mixin 
- 加入 cryptoJS (加解密用)
