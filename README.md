# Vue-Template

## 前言
創建寫 side project 用的模板
[測試網址(停用中)](https://rocmark.github.io/Vue-Template/.)

## 使用
```
  // 開啟 json-server
  npm run json-server

  // 啟動 vue 開發模式
  npm run serve
```

## 功能
- 全域 SCSS 檔案
- 全域 axios
- JSON Server
- Git Commit 模板
- BluePrint 藍圖建立檔案
